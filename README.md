# Email-Campaign
Responsive HTML email template with a clean two-column layout on desktop and single-column view on mobile for better readability. Includes internal linking with UTM source and medium support for campaign tracking, and is open-rate tracking ready. Built using email-safe HTML for broad client compatibility.
